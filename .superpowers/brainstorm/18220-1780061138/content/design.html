Итоговый дизайн

Карта районов Дубая · MapLibre + статика · деплой Netlify

Лестница зума (уровни сменяются)
zoom < 11  🟦 Секторы (9) — обзор всего Дубая
11 – 13  🟩 Районы / districts (14)
zoom ≥ 13  🟧 Коммьюнити (227)
Плавное появление/затухание на границах. Цвет — по сектору.
Взаимодействие
🖱️ Наведение → насыщенная заливка + контур + название
👆 Клик → плавный зум к границам полигона (вглубь)
🔍 Поиск → автодополнение по всем именам → лёт к объекту
🌗 Тема → переключатель Светлая ⇄ Тёмная
🎨 Легенда → цвета 9 секторов
Структура проекта
dubai_areas_map/
├── geojson/                  # исходники (как есть)
│   ├── dubai_sectors.geojson
│   ├── dubai_districts.geojson
│   └── dubai_locations.geojson
├── scripts/
│   └── enrich.mjs            # Turf: проставляет sector в districts/communities
├── public/                   # ← Netlify публикует это
│   ├── index.html
│   ├── style.css
│   ├── app.js                # MapLibre: слои, зум, hover, клик, поиск, тема
│   └── data/                 # обогащённые geojson (вывод enrich.mjs)
├── package.json              # "build": "node scripts/enrich.mjs"
└── netlify.toml              # publish=public, build=npm run build

Разбор по разделам — в терминале. Подтверди или поправь.
